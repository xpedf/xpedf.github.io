Instalación Arch Linux

Publicado 2022-06-11
Por Alvaro Balada Cuartero
3 min de lectura

Arch Linux és una distribución opensource caracterizada por su personalización i configuración. A diferencia de otras distribuciones, Arch linux no cuenta con ningun programa preinstalado, nisiquiera tiene un entorno de escritorio o interfaz gráfica.

Instalaremos Arch linux en una máquina virtual, también és posible instalarlo en la maquina real desde un usb booteable. Empezaremos descargando la ISO de Arch Linux desde la pagina oficial.

Configuración Virtual Box

Una vez descargada la ISO, procedemos a realizar la instalación, en Virtual Box. Antes de iniciar la màquina creada, és muy importante activar la casilla Enable EFI en la pestaña System (Esta opción habilitará el sistema de arranque EFI, sustituyendo al sistema BIOS).

Instalación

Una vez Iniciada la maquina virtual con la ISO de arch linux, cambiaremos la distribución del teclado a Español y comprobaremos el funcionamiento de la red

Crear particiones

(Todas las particiones seran creadas en el disco duro virtual de VirtualBox, las particiones se situan en el directorio /dev/

Crearemos las particiones correspondientes, en mi caso será:

-Partición /dev/sda1 para el gestor de arranque (250M EFI Sytem)

-Partición /dev/sda2 para todo el sistema de archivos (11.4G Linux filesystem)

Una vez indicadas las particiones que queremos crear, seleccionaremos [Write] para aplicar los cambios

1

cfdisk

Formatear particiones

Con los siguientes comandos formatearemos las particiones creadas(sda1–>efi, sda2–>ext4):

1
2
3

mkfs.vfat -F32 /dev/sda1 

mkfs.ext4 /dev/sda2

Montar particiones

Una vez formateadas las particiones, tendremos que montarlas en nuestro sistema actual que esta corriendo en una Live ISO.

Montaremos la partición /dev/sda1 en /mnt/boot/efi y la partición /dev/sda2 en /mnt. (/mnt corresponde a la raíz del nuevo sistema operativo que se instalará en el disco duro)

1
2
3

mount --mkdir /dev/sda1 /mnt/boot/efi

mount /dev/sda2 /mnt

Una vez montadas las particiones, instalaremos todos los paquetes necesarios para Linux en /mnt con el siguiente comando, la instalación tardara un poco.

1

pacstrap /mnt linux linux-firmware base nano grub networkmanager dhcpcd efibootmgr

Generaremos un fstab en /mnt/etc/fstab para aplicar todas las particiones creadas en el nuevo sistema

1

genfstab /mnt >> /mnt/etc/fstab

Zona horaria, hora y distribución del teclado

Entraremos en el sistema de archivos de nuestro nuevo sistema operativo en /mnt con el siguiente comando:

1

arch-chroot /mnt

Una vez dentro, crearemos un link simbòlico para que el archivo de nuestra zona horaria corresponda al archivo /etc/localtime

1

ln -sf /usr/share/zoneinfo/Europe/Madrid /etc/localtime

Descomentaremos la línea correspondiente de nuestra zona en el archivo /etc/locale.gen, en mi caso es_ES.UTF-8 UTF-8

El paso siguiente es configurar la hora, lenguaje y distribución del teclado con estos comandos, la hora se aplicarà basándose en las configuraciones anteriores

1
2
3
4
5
6
7

locale-gen

hwclock-w

echo KEYMAP=es > /etc/vconsole.conf

echo LANG=es > /etc/locale.conf

Instalaremo el gestor de arrangue GRUB, para que se pueda iniciar nuestro sistema operativo cuando iniciemos el pc o máquina virtual:

Contraseñas i usuarios

Muy importante crear una nueva contraseña para el usuario root i crear un nuevo usuario con los siguientes comandos:

1

passwd

Creación de nuevo usuario

1
2

useradd alvaro -m
passwd alvaro

Reiniciaremos el sistema y se iniciará directamente el arch linux instalado en nuestro disco duro

Una vez dentro, configuraremos el hostname (en mi caso será ALVARO) y el Networkmanager para habilitar el internet

1
2
3
4

echo ALVARO >> /etc/hostname

systemctl start NetworkManager
systemctl enable Networkmanager

Con este último paso, ya tendriamos Arch Linux instalado en nuestro disco duro
